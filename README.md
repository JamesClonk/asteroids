Asteroids
=========

OpenGL Asteroids in Golang..

### Installation instructions

* `pacman -S glew`
* `pacman -S glfw`
* `pacman -S geos`


* `go get github.com/go-gl/gl`
* `go get github.com/go-gl/glfw3`
* `go get github.com/paulsmith/gogeos/geos`
* `go get github.com/JamesClonk/asteroids`
* `go build`

### Run

`./asteroids`
